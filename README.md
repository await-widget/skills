# Await Widget Skills

Await is an iOS widget workshop app, where you can build any widget. This repository is the installable `await-widget` agent skill.

## Links

- [Download Await](https://apps.apple.com/app/id6755678187)
- [Skill Instructions](await-widget/SKILL.md)
- [Privacy Policy](PRIVACY.md)
- [Feedback](https://github.com/await-widget/skills/issues)

## Install As A Skill

Install the `await-widget` skill via [`npx skills`](https://github.com/vercel-labs/skills):

```bash
# Project-scoped
npx skills add await-widget/skills

# User-scoped
npx skills add await-widget/skills -g
```

Restart your agent after installing.
